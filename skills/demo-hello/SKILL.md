# Demo Hello Skill

这是一个演示 skill，展示了 lx-skills 仓库中 skill 的基本结构。

## 功能

当被调用时，此 skill 会：
1. 输出友好的问候语
2. 提供关于如何使用 skills 的简单提示

## 使用方法

在支持的 AI 编程工具中，直接调用此 skill：

```
使用 demo-hello skill
```

## 示例输出

```
你好！这是来自 demo-hello skill 的问候。

提示：你可以通过创建类似的 skill 来扩展 AI 工具的能力。
查看 docs/skill-format.md 了解如何创建自己的 skill。
```

## 扩展

这个 skill 可以作为创建新 skill 的模板。复制此目录结构，修改 SKILL.md 的 front matter 和内容即可。
